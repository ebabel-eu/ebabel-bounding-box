# ebabel-bounding-box
Calculate a 3D bounding box for a THREE.js mesh.

## Instructions for Module Usage in your project/game

### Install
```
npm i --save ebabel-bounding-box
```

### Usage
```
const boundingBox = require('ebabel-bounding-box');

bounding-box(THREE, mesh);
```
